TechNova 2026 - Event Registration Website

A modern and responsive event registration website created as part of my Web Development Internship.

📌 Overview

TechNova 2026 is a fictional inter-college technical fest registration website. It features an attractive landing page with event details and a user-friendly registration form.

✨ Features

- Modern UI with a gradient background
- Responsive design
- Event information section
- Student registration form
- Clean and organized layout

🛠️ Built With

- HTML5
- CSS3

Demo:
www.kanak1507.github.io/TechNovaRegistrationForm

🚀 Getting Started

1. Clone the repository.
2. Open "index.html" in your web browser.
3. Explore the website.

- Designing clean and user-friendly interfaces
